# Production Checklist

## Before Recording

- Open PMO dashboard in owner/read-only view.
- Hide account secrets and tokens.
- Keep live trading locked on screen if visible.
- Open TradingView or chart visual only when relevant.
- Prepare 3 to 5 screen sections to show.
- Confirm disclaimer is included.

## Recording Structure

1. Hook in the first 10 seconds.
2. Disclaimer.
3. Show the main dashboard or chart.
4. Explain one specific finding.
5. Show the evidence.
6. Explain what changes because of the evidence.
7. Close with the next video topic.

## Visual Rules

- Use big zooms on dashboard numbers.
- Blur account IDs, API keys, tokens, and private paths.
- Do not show broker credentials.
- Use red boxes/arrows for one key point per scene.
- Avoid cluttered full-screen dashboard shots for too long.

## Audio Rules

- Keep voice calm and direct.
- Avoid promising profits.
- Say "paper trading" clearly.
- Use "the data shows" only when a real audit or journal result is visible.

## Upload Package

- Title
- Thumbnail text
- 2-line hook description
- Disclaimer
- Tags
- Pinned comment
- End-screen prompt to next video
